# exp_fileLooper
file looper from beamtime. Thanks to xaratustrah@github

## Prerequisites
- `Python 3`
- `multiprocess`, `matplotlib`
- [`psd-analysis`](https://github.com/NanaVan/psd-analysis)

## Usage
1. configure your own setting for assigned data file at the file `looper_cfg_data.toml`.
2. processing your looper at the terminal 
```Python
> python3 looper_data.py --config looper_cfg_data.toml
```
